# Icon Images

Place your custom icon images here.

## File Names:
- `about.png` - About Me icon
- `projects.png` - Projects icon
- `skills.png` - Skills icon
- `contact.png` - Contact icon
- `blog.png` - Blog icon
- `gallery.png` - Gallery icon

## Recommended Specifications:
- **Size**: 48x48 pixels (or square aspect ratio)
- **Format**: PNG (with transparency) or JPG
- **Style**: Retro/pixelated style works best with the theme
- **Background**: Transparent PNG recommended for best results

## Notes:
- If an icon image is missing, a placeholder will be shown automatically
- Icons should be square (1:1 aspect ratio) for best display
- Pixelated/retro style icons match the overall aesthetic
